# Voltaserve UI

Install dependencies:

```shell
bun i
```

Run for development:

```shell
bun run dev
```

Lint TypeScript code:

```shell
bun run lint
```

Format TypeScript code:

```shell
bun run format
```

Lint Go code:

```shell
golangci-lint run
```

Build Docker Image:

```shell
docker build -t voltaserve/ui .
```
