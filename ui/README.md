# Voltaserve UI

## Getting Started

Install [Node.js 18.x](https://nodejs.org).

Enable [pnpm](https://pnpm.io):

```shell
corepack enable
```

Install dependencies:

```shell
pnpm install
```

Run for development:

```shell
pnpm run dev
```

Build Docker Image:

```shell
docker build -t voltaserve/ui .
```
