# Voltaserve UI

Install dependencies:

```shell
npm i
```

Run for development:

```shell
npm run dev
```

Build for production:

```shell
npm run build
```

Run for production:

```shell
npm run .
```

Lint TypeScript code:

```shell
npm run lint
```

Format TypeScript code:

```shell
npm run format
```

Lint Go code:

```shell
golangci-lint run
```

Build Docker Image:

```shell
docker build -t voltaserve/ui .
```
