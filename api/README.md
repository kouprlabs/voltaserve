# Voltaserve API

## Getting Started

Install [Golang](https://go.dev/doc/install).

### Build and Run

Run for development:

```sh
go run .
```

Build binary:

```sh
go build .
```

Build Docker image:

```sh
docker build -t voltaserve/api .
```
