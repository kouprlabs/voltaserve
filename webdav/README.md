# Voltaserve WebDAV

Install dependencies:

```shell
bun i
```

Run for development:

```shell
bun run dev
```

Run for production:

```shell
bun run start
```

Lint code:

```shell
bun run lint
```

Format code:

```shell
bun run format
```

Build Docker image:

```shell
docker build -t voltaserve/webdav .
```
