# Voltaserve WebDAV

## Getting Started

We assume the development environment is setup as described [here](../DEVELOPMENT.md).

### Build and Run

Install dependencies:

```shell
pnpm install
```

Run for development:

```shell
pnpm run dev
```

Run for production:

```shell
pnpm run start
```

Build Docker image:

```shell
docker build -t voltaserve/webdav .
```
